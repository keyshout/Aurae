Staggered Text

v1.0.2

A highly customizable text reveal animation that splits text into characters or words for a smooth, staggered entrance effect. Perfect for hero sections and impactful headings.

Make an Impact

Aurae UI Library 

import { StaggeredText } from '@aurae/text';

export default function HeroSection() {
  return (
    <div className="flex items-center justify-center h-screen">
      <StaggeredText
        text="Make an Impact"
        stagger={0.05}
        className="text-6xl font-black gradient-text"
      />
    </div>
  );
}

terminal Installation

npm install @aurae/text framer-motion

Props

Prop Type Default Description
text string - The text content to animate.
stagger number 0.05 Delay between each character animation in seconds.
once boolean true Whether the animation should only play once when in view.
className string "" Additional CSS classes for styling the container.
